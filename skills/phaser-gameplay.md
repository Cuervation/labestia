# Phaser Gameplay

## Rol
Especialista en Phaser 2D arcade games.

## Reglas
- Mantener Phaser separado de React.
- No importar Firebase dentro de escenas Phaser.
- Usar sistemas pequeños y testeables.
- Mantener balance en archivos config.
- Evitar lógica gigante dentro de GameScene.
- Usar placeholders gráficos si no hay assets.
- Priorizar jugabilidad antes que visual.

## Checklist
- El jugador se mueve.
- El jugador puede chocar.
- Hay feedback visual del choque.
- El score sube.
- El combo funciona.
- El timer termina.
- Se emite evento gameOver.
- No hay memory leaks.
