# Game Spec — La Bestia

Juego arcade top-down de autos. Partidas de 90 segundos. El jugador maneja una camioneta roja de fletes llamada `La Bestia`, suma puntos chocando autos y encadena combos dentro de una ventana corta.

## Loop

1. Login.
2. Jugar 90 segundos.
3. Chocar autos.
4. Encadenar combo multiplicador.
5. Guardar puntaje.
6. Mostrar ranking global.
