# Game Balance — La Bestia

Valores iniciales. No son definitivos.

| Parámetro | Valor |
|---|---:|
| Duración | 90 s |
| Score por choque | 100 |
| Ventana combo | 2 s |
| Multiplicador inicial | x1 |
| Multiplicador máximo inicial | x8 |

## Principios

- Combo debe premiar riesgo.
- Score debe ser legible.
- Balance vive en config, no en escenas.
